# 欢迎来到我的博客

这是我的第一篇博客文章！很高兴你能来到这里。

## 技术栈

这个博客使用了以下技术：

- **Next.js 15** - React 全栈框架
- **TypeScript** - 类型安全的 JavaScript
- **Tailwind CSS** - 实用优先的 CSS 框架
- **Contentlayer** - 基于文件的内容管理系统
- **MDX** - 支持 JSX 的 Markdown

## 功能特性

### 代码高亮

支持多种编程语言的语法高亮：

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`
}

console.log(greet("World"))
```
